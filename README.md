
## 1) Null এবং Undefined এর মধ্যে পার্থক্য
- **Null:** প্রোগ্রামারের দ্বারা খালি মান নির্ধারণ।  
- **Undefined:** ভেরিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো মান দেওয়া হয়নি।  
- **মূল পার্থক্য:** Null স্পষ্ট খালি মান, Undefined হলো স্বয়ংক্রিয়ভাবে দেওয়া মান।

---

## 2) map() ফাংশনের ব্যবহার এবং forEach থেকে পার্থক্য
- **map():** প্রতিটি উপাদানের উপর ফাংশন প্রয়োগ করে এবং **নতুন অ্যারে** রিটার্ন করে।  
- **forEach():** প্রতিটি উপাদানের উপর ফাংশন প্রয়োগ করে, কিন্তু **কিছু রিটার্ন করে না।**  
- **মূল পার্থক্য:** map() → নতুন অ্যারে, forEach() → কোন রিটার্ন নেই।

## 3) == এবং === এর মধ্যে পার্থক্য
- **== :** মান চেক করে, টাইপ কনভার্শন হয়।  
- **=== :** মান এবং টাইপ উভয় চেক করে, টাইপ কনভার্শন হয় না।  
- **উদাহরণ:** `5 == '5' → true`, `5 === '5' → false`

## 4) API ডেটা fetch করার ক্ষেত্রে async/await এর গুরুত্ব
- Async/await দিয়ে অ্যাসিঙ্ক্রোনাস কোডকে সহজে সিনক্রোনাসের মতো লেখা যায়।  
- কোড পড়তে সহজ হয় এবং error handling সহজ।  
- উদাহরণ: `await fetch()` দিয়ে API ডেটা সহজে পাওয়া যায়।

## 5) JavaScript এ Scope এর ধারণা
- **Global Scope:** ফাংশন বা ব্লকের বাইরে, পুরো প্রোগ্রামে অ্যাক্সেসযোগ্য।  
- **Function Scope:** শুধুমাত্র ফাংশনের ভিতরে অ্যাক্সেসযোগ্য।  
- **Block Scope:** শুধুমাত্র নির্দিষ্ট ব্লকের ভিতরে অ্যাক্সেসযোগ্য (`let`/`const`)।
